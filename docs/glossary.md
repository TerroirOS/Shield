# Glossary

## Appeal Case

A structured review record opened when a decision is contested or requires human adjudication.

## Basis Risk

Mismatch between index-triggered outcome and realized participant loss.

## Climate Event

A normalized observation used as input for rule evaluation.

## Decision Artifact

A recorded output from evaluation/simulation containing amount, status, and governance metadata.

## Evidence Reference

Pointer to source documents or datasets supporting an event or decision.

## Program

A bounded resilience-response configuration with explicit rules, thresholds, and governance roles.

## Rule Version

Immutable identifier for the decision logic used during trigger evaluation.

## Shield

Planned TerroirOS resilience module for climate-risk visibility, event tracking, and transparent response workflows.
