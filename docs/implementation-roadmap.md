# Implementation Roadmap

Status: Planned / not launched.

## Phase 0 - Documentation and Specs (Current)

Exit criteria:

- Core docs and spec pack are published.
- CI checks pass for markdown, links, OpenAPI, and schemas.
- Governance and basis-risk pathways are explicitly documented.

## Phase 1 - Data Ingestion Prototype

Exit criteria:

- Event ingestion and retrieval endpoints implemented in non-production environment.
- Source-admissibility checks and envelope validation enforced.
- End-to-end event traceability demonstrated.

## Phase 2 - Trigger Engine Prototype

Exit criteria:

- Rule-versioned deterministic evaluation implemented.
- Decision artifacts include governance flags.
- Reproducibility tests pass for fixed fixtures.

## Phase 3 - Review and Appeal Workflow

Exit criteria:

- Manual review state machine implemented.
- Appeal case recording and resolution metadata available.
- Audit logs provide reviewer traceability.

## Phase 4 - Public Reporting Surface

Exit criteria:

- Public reporting endpoint publishes non-sensitive outputs.
- Privacy boundary tests pass.
- Operational and governance documentation updated for pilot readiness.
