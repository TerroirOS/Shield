# Overview

Terroir Shield is the planned resilience extension of TerroirOS.
It builds on provenance and evidence primitives to support climate-risk visibility, event tracking, and transparent response decisioning.

Status: Planned / not launched.

## Problem Context

Origin-sensitive agricultural systems face both trust fragmentation and climate shocks.
Without verifiable enrollment, event traceability, and published decision logic, support distribution becomes slow and dispute-prone.

## Module Intent

Shield is intended to:

- Ingest program-relevant climate events.
- Evaluate rule-based triggers with auditable metadata.
- Produce transparent decision artifacts.
- Publish public reporting without exposing protected participant data.
- Support manual review and appeal for basis-risk or data-quality exceptions.

## Non-positioning

Shield documentation does not position the module as a replacement for regulated insurance regimes.
The current scope is resilience workflow support with explicit governance controls.
